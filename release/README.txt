AutoPiar

1. Запустите AutoPiar.exe
2. Введите адрес сервера лицензий и ключ.
3. После успешной проверки откройте API настройки и введите Telegram API ID / API HASH.

Все настройки, лицензия и Telegram-сессия хранятся в:
%APPDATA%\AutoPiar

Если нужно сбросить настройки, закройте программу и удалите эту папку.
